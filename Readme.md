## Built With ✨

- JavaScript ECMA6
- Node.js v13.2.0
- Jest 25.1.0

To get a local copy up and running follow these simple example steps.

### Prerequisites
* Node.js installed
You will need Node.js up to v8

* System dependencies
Windows/ Mac


### Install
Clone project and run
`npm install`
`npm run test`
